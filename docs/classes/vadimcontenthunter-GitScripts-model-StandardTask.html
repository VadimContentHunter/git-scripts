Tags

Return values
int —

Возвращает значение которое вернул скрипт. Вернет:

  • 0 - в случае успешного выполнения
  • -1 - в случае если задача не обладает статусом WAITING
  • -2 - в случае если скрипт не вернет значение
  • -3 - в случае если по какой либо причине скрипт не выполниться

StandardTask.php : 331

Выполняется после выполнения метода execute.

public setWhenExecuteValue(callable $_function) : StandardTask

С начало выполниться функция установленная в этом методе, потом вернется значение execute. В качестве 1 аргумента, функция принимает текущий объект, а 2 результат выполнения метода execute Результат пользовательской функции будет не обработан.
